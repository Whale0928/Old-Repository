        선택자 { color : 색상명(영문),16진수 숫자(#fff,#ffffff)
                rgb(255,255,255)|rgba(255,255,255,1)      rgba에서 a는 투명도를 의미한다
                hsl(360,100,100)|rgba(255,255,255,1)  색조,채도,밝기 , hsla에서 a는 투명도(0:투명,1:불투명)
                6개의 색상 지정 방법은 색과 관련된 모든 css 속성에서 사용가능    
            }
  • 색상명
  • 16진수
  • rgb로 지정
  • rgba로 지정
  • hsl로 지정
  • hsla로 지정

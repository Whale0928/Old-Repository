        form 태그 : 
            내부에 작성된 input 태그의 값을 지정된 서버 혹은 페이지로 제출(전달)하는 역할

            ** 이 때 어떤 input 태그에 작성된 값이 제출 된건지 알 수 있도록 
                input 태그에 name 속성을 '반드시' 작성한다.


            ** form 태그 속성
            - action : input 태그 값을 전달할 서버 혹은 페이지의 주소를 작성하는 속성.
            
            - method : 데이터 전달 방식 지정 속성 (Get / Post)

        GET 방식 : 요청하는 주소 뒤에 전달하려는 값 (input 태그의 값)을
                    문자열 형태로 붙여서 서버로 전달하는 방식
            
        -장점 : 단순함,캐싱 가능(요청 저장 가능==북마크/즐겨찾기) 
        -단점 : 보안 취약함(눈에 다 보임)

        ** GET 방식이 form 태그, a태그 , 주소창 직접입력 것들이 GET방식

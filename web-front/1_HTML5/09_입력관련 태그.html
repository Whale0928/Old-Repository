type = "text"

        -한줄짜리 문자열(텍스트)를 입력 할 수 있는 입력 상자
        -input 태그의 type 기본값으로 생략 가능

        -size : 입력 상자의 크기

        -placeholder : 입력 상자에 작성된 내용을 설명하는

form 태그

        -div와 같은 영역(block 형식)
         + 내부에 작성된 input 태그의 값을 서버로 전달하는 역활

        [속성]
        1)action : 내부에 작성된 값을 전달할 서버 또는 페이지의 주소를 작성
        2)method : 서버로 데이터 전달하는 '방식'을 지정하는 속성
        3)name   : form 태그의 이름을 지정하여 각각의 form 태그 구분(js 사용)
        4)target : action에 지정된 주소를 새탭/현재탭 중 어디서 열지 지정

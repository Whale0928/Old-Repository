02_목록 관련 태그

ul tag : 순서 없는 목록을 작성하는 태그(영역)

li(List Item) tag : 목록에 들어가는 내용을 작성하는 태그

  • 스파게티
  • 햄버거
  • 깁밥천국
            기본 값 : 1 부터 1씩 증가하는 숫자
            
            ol태그의 type 속성 * 

            type ="a" -->영어 소문자 순서
            type ="A" -->영어 대문자 순서
            type ="i" -->로마 소문자 순서
            type ="I" -->로마 대문자 순서
            
            +추가 속성
            start ="5"->  순서 5부터 시작
            reversed  ->  순서 반대로

ol(Ordered List) tag :순서 있는 목록을 작성하는 태그
